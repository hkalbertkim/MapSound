# Licensing Migration Summary

## What Changed

MapSound now uses a Business Source License 1.1 (BSL 1.1) model. The repository now includes a root [LICENSE](/Users/albertkim/02_PROJECTS/14_mapsound/LICENSE) file customized for MapSound, a commercial licensing explainer in [COMMERCIAL.md](/Users/albertkim/02_PROJECTS/14_mapsound/COMMERCIAL.md), an updated license section in [README.md](/Users/albertkim/02_PROJECTS/14_mapsound/README.md), and short source-availability headers in major first-party source files.

## Why BSL Was Chosen

BSL 1.1 supports public source visibility and community engagement while placing clear limits on unlicensed commercial exploitation. This fits MapSound's goal of being friendly to fans, researchers, and contributors without allowing third parties to operate competing commercial or hosted offerings without permission.

## Allowed Uses

- personal use
- non-commercial use
- educational use
- research use
- internal evaluation use

## Uses Requiring a Commercial License

- commercial use
- commercial SaaS or hosted production use
- paid API offerings
- embedding MapSound into a commercial platform
- internal enterprise production deployment
- white-label or reseller usage
